# 和田のSchematic置き場

フィリー和田や関連人物によって製作されたschematicファイルを配布しています。

# 環境について

特筆しない限り、

- Minecraft 1.7.10
- Forge 10.13.4.1614
- HariboteAirCraft 172v3
- MCEdit Unified v1.5.6.0

で開発しています。

この環境で動作しない場合、ツイッター・Discordにお問い合わせください。

# 利用について

ファイルの中には、ボタンを押せば射撃できるものから、制御装置がなければ動作しないような技術者向きのものも含まれており、特にJunkフォルダの中には動作しない前提のものも入っています。
利用方法が分からない場合や、技術的な質問等は、ツイッター・Discordにてお問い合わせください。
利用している技術自体には権利が発生しないはずなので、分解して参考にするなどはご自由にどうぞ。
使い方によっては、ワールドをクラッシュさせてしまう場合があります。設定等により改善する場合が多いので、もしクラッシュするようになってしまった場合は、ツイッター・Discordにお問い合わせください。
